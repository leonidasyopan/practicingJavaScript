Notas da Aula 05

Varíaveis podem ser declaras usando:

  • var
  • let

Temos 3 tipos de variáveis:

  • Numbers
  • Strings
  • Booleans

As strings podem ser atribuidas usando uma das 3 opções:

  • "Meu nome"
  • 'Meu nome'
  • `Meu nome`

Outros Data Types

  • Null
  • Undefined
  • Object
  • Function

O comando typeof é muito útil pra ser qual o Data Type de uma variável.
